互联网 + 乡村旅游 = 村游，专注乡村旅游细分市场的全国性平台。

村游创办于2011年，战略总部设在北京，技术和运营中心设在西安，已在北京、陕西、湖北、湖南、福建、江西、重庆、贵州、云南、四川10个省设立省级运营中心。

村游历时5年，深入10多个省，400多个县，过万家商户，探索实践乡村旅游+互联网，实现两部手机打通商户和用户，连接城市和乡村。以乡村旅游+农产品为特色，提出后备箱工程，创立旅游驱动型农村电商模式。围绕商户、用户、政府及上游供应商，以“村游网”为基础，开发了“村游管家”、“村游学院”、“村游＋”，构成四大服务体系。

村游独特的商业模式和实战型的团队，得到资本市场青睐，2014年底，获得高德地图创始人成从武先生的鑫澄资本A轮2000万融资。

目前，村游已和各省旅游局、扶贫办、商务厅等政府机构建立密切合作，合作收录商户过万家，年交易量过亿元。

村游历程：
  • 2011年，创办村游项目；
  • 2012年，成为陕西省旅游局乡村旅游官方网站；
  • 2013年，成为湖北省旅游局制定乡村旅游电商网站，多次受邀给各地旅游局、 经营户做培训；
  • 2014年，成为重庆市扶贫办战略合作单位；
    成为国务院扶贫办农村电商试点企业；
    完成A轮2000万融资，成立西安村游网络技术有限公司；
  • 2015年，成为西安市商务厅电商示范企业；
  • 2016年，获得陕西省商务厅商业云端黑马奖；
    当选中国扶贫志愿服务促进会旅游扶贫联盟理事单位。和各地旅游局合作，在全国开展“互联网+乡村旅游”现场培训会100余场。
